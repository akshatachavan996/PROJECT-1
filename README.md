# 📘 LabMate

LabMate is a web-based platform that helps students access Physics and Chemistry experiment videos along with solved PDFs in one place. It is designed to simplify practical learning with a clean and user-friendly interface.

---

## 🚀 Features
- 📂 Subject-wise organization (Physics & Chemistry)
- 🎥 Access to experiment videos
- 📄 Solved PDF materials available
- 💡 Clean and simple UI for easy navigation

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📌 Project Goal
To make practical learning easier and more accessible for students by providing all experiment-related resources in one platform.
